Live Ausdruck von Kundenfeedback
Wenn Sie über einen Epson TM-T20 Drucker besitzen, kann das Kunden-Feedback aus Ihrem Airfy WLAN sofort ausgedruckt werden. Die Kollegen vor Ort können dadurch direkt reagieren. Der handelsübliche Epson TM-T20 Drucker wird automatisch von Airfy erkannt. Der Drucker muss sich im selben Netzwerk die Ihre Airfy Nodes befinden. Drücken Sie bitte auf den nachfolgenden Button um nach allen aktuellen Druckern zu suchen und bestimmen Sie dann, auf welchem Gerät der ausdruck erfolgen soll. Wenn ein oder mehrere Drucker erfolgreich erkannt wurde, sehen Sie die Drucker unter dem Aktuaulisiren butten. Sollte dies nicht der Fall sein, konnten wir keine Drucker finden.
Erfolg. Drucker gefunden
Wenn Sie mehrere Drucker besitzen, drücken Sie bitte die Funktion "Testdruck" an. Es erfolgt dann auf jedem Ihrer Drucker, ein Ausdruck mit dem Inhalt: Drucker 1 Standort XYZ, Drucker 2 usw. Dadurch können Sie Ihre Drucker identifizieren und über den "ON / OFF" Regeler bestimmen, auf welchem Drucker das Feedback erfolgen soll. Sie können das Feedback natürlich auch auf mehreren Druckern ausgeben. Damit wir Ihren Papierverbrauch und die Umwelt schonen, erfolgt der Ausdruck nur von Feedback das länger als 5 Zeichen ist.
