General Installation
  1. Simple requirements of your existing internet router and network:
    1. Have DHCP / Automatic IP-Address active;
    2. Turn off your firewall. Learn more;
    3. Install them on a high place;
  2. Use the wall mounting;
  3. The antennas must be vertical. Don't lay down the system horizontally;
  4. If you can use a network cable at all nodes, try to just have one wall between each node or maximum two walls. Walls with to much metal and stone will result in a bad wifi quality;
  5. Plugin a network cable and use only the airfy power supply. Follow the instructions inside the Airy Node packaging. If you missed them, click here to download a PDF;
